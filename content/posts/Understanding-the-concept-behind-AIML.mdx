## What is AIML?

AIML or Artificial Intelligence Markup Language is an XML base language used for creating chatbots(AI), it is always saved with the extension (.aiml).

The tag always used in AIML used to specify a new function is the `category` tag which has two other tags running under it that is the `pattern` tag and the `template` tag. Inside the template tage we could also have other tags inbuilt ‚ÄúDepends on the type of BOT you are building‚Äù eg `set`, `br/`, `srai`‚Ä¶etc

But note something just as in HTML or any other programming Language or Scripting Language Even Markup Languages have a closing tag, here in AIML there is really not much a difference, the closing tags are `/category`, `/template`, `/srai>`, `/set`‚Ä¶etc
